# DIPG Examples

These examples demonstrate workflows around DIPG Safety Gym and related medical-safety evaluation setups.

They are **integration examples**, not the canonical introduction to OpenEnv APIs.

Use them when you want to see how external evaluation or training workflows can live alongside OpenEnv.

If you are looking for the standard OpenEnv patterns first, start here instead:

- `docs/source/index.md`
- `docs/source/auto_getting_started/` in the rendered docs site
- `docs/source/mcp-environment-lifecycle.md`
- `docs/source/simulation-vs-production.md`
